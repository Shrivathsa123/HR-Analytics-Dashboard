
# 📊 HR Analytics Dashboard

## 🔹 Overview

This project is an interactive **HR Analytics Dashboard** built using **Microsoft Power BI** to analyze employee data and uncover insights related to attrition patterns. It helps organizations make data-driven HR decisions by identifying key factors influencing employee turnover.

---

## 🎯 Objectives

* Analyze **employee attrition trends**
* Identify **key factors affecting employee turnover**
* Understand employee distribution based on **age, salary, and education**
* Provide insights for **improving HR decision-making**

---

## 📈 Key Insights

* Highest attrition is observed among employees aged **26–35**
* Employees with **Life Sciences** and **Medical** backgrounds show higher attrition
* Majority of attrition occurs in the **salary range up to 5K**
* Employees with **fewer years at the company** are more likely to leave
* **Laboratory Technicians** and **Research Scientists** experience higher attrition

---

## ⚙️ Features

* KPI cards for:

  * Employee Count
  * Attrition Count & Rate
  * Average Age, Salary, and Tenure
* Attrition analysis by:

  * **Education** (Donut Chart)
  * **Age Group** (Bar Chart)
  * **Salary Range** (Bar Chart)
  * **Years at Company** (Line Chart)
  * **Job Role** (Bar Chart)
* **Gender-based attrition comparison**
* **Department-level filtering**

---

## 🚀 How to Use

1. Download the `.pbix` file from this repository
2. Open it in **Power BI Desktop**
3. Use filters to explore different HR insights
